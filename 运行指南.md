# 运行指南

## 1. 环境要求

- JDK 17+
- Maven 3.8+
- MySQL 8+
- Node.js 18+
- npm 9+

## 2. 初始化数据库

```bash
cd backend
mysql -u root -p < 数据库初始化脚本/init.sql
```

如果本机 `mysql` 命令不在 PATH，可在 `backend` 目录使用 Maven 初始化：

```bash
mvn -q -DskipTests test-compile exec:java "-Dexec.mainClass=com.hospital.registration.DbInitializer" "-Dexec.classpathScope=test" "-Dexec.args=你的MySQL密码"
```

当前示例 MySQL 用户名和密码为 `root / 123456`。如果你的本机配置不同，请修改：

```text
backend/src/main/resources/application.yml
```

也可以在启动后端前设置环境变量：

```bash
DB_USERNAME=root
DB_PASSWORD=你的MySQL密码
JWT_SECRET=请替换为至少32位的JWT签名密钥
```

Windows PowerShell 示例：

```powershell
$env:DB_USERNAME="root"
$env:DB_PASSWORD="你的MySQL密码"
$env:JWT_SECRET="replace-with-your-32-char-jwt-secret"
```

## 3. 运行后端测试

```bash
cd backend
mvn test
```

## 4. 启动后端

```bash
cd backend
mvn spring-boot:run
```

后端地址：

```text
http://localhost:8080
```

## 5. 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端地址：

```text
http://localhost:5173
```

## 7. 推荐演示流程

1. 使用患者账号登录。
2. 进入“就诊人管理”，确认或新增就诊人。
3. 回到“科室挂号”，选择科室和医生。
4. 在医生详情页选择可预约号源。
5. 选择就诊人并提交预约。
6. 在预约成功页查看预约号和注意事项。
7. 进入“我的预约”，测试 30 分钟内取消预约。
8. 使用管理员账号登录，查看统计面板并维护排班。
